# Frontend - DCA Trading Bot

Веб-интерфейс на Vue 3 + Tailwind CSS для торгового бота.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте браузер: `http://localhost:3000`

## Сборка

```bash
npm run build
```

## Технологии

- Vue 3 (Composition API)
- Vite
- Tailwind CSS
- Axios
